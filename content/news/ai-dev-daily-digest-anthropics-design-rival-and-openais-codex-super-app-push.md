## OpenAI's Codex Gains Background PC Control and In-App Browser
![A user works with financial data in Codex](https://cdn.arstechnica.net/wp-content/uploads/2026/04/Codex-screenshot-640x360.jpg)
**OpenAI** has rolled out a substantial update to its **Codex** desktop app, introducing a suite of new features that significantly enhance its capabilities. Most notably, **Codex** can now perform tasks on your PC in the background. **OpenAI** states that this functionality allows the AI to use all applications on a computer by seeing, clicking, and typing with its own cursor, all without interfering with a user's ongoing work on the desktop. This means multiple agents can operate in parallel on a **Mac**, proving beneficial for developers iterating on frontend changes, testing apps, or working with applications lacking an API.

Beyond background operation, **Codex** also gains the ability to schedule work, waking itself up hours, days, or even weeks in advance to perform predefined tasks. An integrated in-app web browser is another key addition, allowing users to visually assess **Codex's** web-related work and provide feedback directly on specific parts of a page, akin to existing web design collaboration tools. The update further incorporates **gpt-image-1.5** for image generation, enabling the creation of mockups and other visual assets within the application.

> **OpenAI's Codex** can now seamlessly interact with and control desktop applications in the background, marking a significant step towards autonomous computing.

## OpenAI's Latest Codex Update Lays Groundwork for "Super App" Vision
![OpenAI's latest Codex update builds the groundwork for its upcoming super app - Engadget](https://s.yimg.com/ny/api/res/1.2/c3iOFJpUlr.M6dUIiPidBw--/YXBwaWQ9aGlnaGxhbmRlcjt3PTEyNDI7aD02OTk-/https://s.yimg.com/uu/api/res/1.2/homOSNkaJc9teyJ4t87vIg--~B/aD0yMTYwO3c9MzgzOTthcHBpZD15dGFjaHlvbg--/https://d29szjachogqwa.cloudfront.net/images/user-uploaded/sidebar_with_artifact_panel_0833.png)
**OpenAI** has released a major update to its **Codex** application, signaling the initial stages of its ambitious "super app" project. While the full super app, which aims to combine **ChatGPT**, the **Codex** coding agent, and the **Atlas** web browser, isn't launching today, this update offers developers a preview of **OpenAI's** future direction. Thibault Sottiaux, head of **Codex**, stated that **OpenAI** is "building the super app out in the open," with a clear intention to broaden its audience beyond developers in the future.

The updated **Codex** introduces multi-purpose AI agents capable of interacting with other applications on a PC, allowing users to either specify a program or let the AI select the most suitable tool. **OpenAI** highlights its proprietary "secret sauce" that enables agents to run apps in the background without impacting system performance, facilitating collaborative work between user and AI. Additionally, the release includes **111 new plugins** that integrate skills, app connections, and model context, expanding **Codex's** ability to gather information and utilize developer tools. A built-in browser with a commenting system and integrated **gpt-image-1.5** image generation further solidify its comprehensive capabilities.

> **OpenAI's Codex** update is a strategic move, showcasing the foundational elements of its upcoming "super app" designed to provide developers with highly proactive and versatile AI agents.

## OpenAI’s Codex Update Takes Direct Aim at Anthropic’s Claude Code
![Screenshot 2026-04-16 at 13.01.11](https://platform.theverge.com/wp-content/uploads/sites/2/2026/04/Screenshot-2026-04-16-at-13.01.11.png?quality=90&strip=all&crop=6.6118941326531%2C5.8356559824806%2C84.116836734694%2C89.31166633486&w=2400)
**OpenAI** has significantly upgraded its agentic coding and development system, **Codex**, with a suite of features designed to intensify its competition with rivals, particularly **Anthropic’s Claude Code**. The update enables **Codex** to operate desktop applications on **macOS**, including working in the background without interrupting user activities. This functionality is described as highly beneficial for developers, aiding in testing and iterating on frontend changes, app testing, and interacting with applications that lack an API. The rollout for **macOS** users signed in with **ChatGPT** begins today, with EU users and other operating systems to follow.

Furthermore, **Codex** now supports image generation and iteration using **gpt-image-1.5**, and includes new plugins for popular developer tools such as **GitLab**, **Atlassian Rovo**, and **Microsoft Suite**. An in-app browser allows users to provide direct instructions to the agent through comments on web pages. The update also introduces enhanced automation capabilities, allowing users to reuse conversation threads and enabling **Codex** to schedule and automatically perform future tasks. A preview of an opt-in memory feature will allow **Codex** to recall personal preferences, corrections, and previously gathered information, aiming to expedite future tasks and improve output quality without extensive custom instructions.

> **OpenAI's Codex** is making a bold move against **Anthropic’s Claude Code** by introducing extensive desktop control, advanced image generation, broad plugin support, and critical memory features to streamline developer workflows.
